# Gemini Logo Skill

Generate consistent WoW-style pixel art icons and textures using Gemini's image generation capabilities.

## Usage

Invoke this skill when you need to generate:
- UI icons (32x32px, 48x48px, 64x64px)
- Tileable textures (128x128px, 256x256px)
- Decorative elements (corners, dividers, frames)

## Style Guide

All generated assets should follow the **Classic World of Warcraft (2004-2010)** aesthetic:

### Color Palette
- **Primary**: Warm browns (#4a3c2a, #6b5a42), tans (#8b7355, #a08060)
- **Accents**: Gold (#c9a227, #e8c84a), bronze (#b87333)
- **Shadows**: Deep browns (#1a1510, #2a2015)
- **Highlights**: Pale gold (#f4d35e), cream (#f4e4c1)

### Visual Characteristics
- Hand-painted appearance with visible brushstrokes
- Warm, earthy tones with metallic gold accents
- Subtle wear and weathering on stone/metal
- Soft inner glow on magical elements
- Clear silhouettes optimized for small sizes

## Generation Workflow

1. **Batch similar assets** - Generate all icons in one session for consistency
2. **Review for style match** - Ensure all assets feel cohesive
3. **Optimize file sizes** - Compress PNGs without quality loss
4. **Test at actual size** - Verify clarity at intended display size

## Fallback Strategy

If Gemini generation is unavailable or slow:
1. Use CSS gradients and shadows to approximate textures
2. Use emoji or Unicode symbols as temporary icon placeholders
3. Create simple SVG icons with the WoW color palette
